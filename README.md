# 🚗 Car Rental Manager

A modern car rental management dashboard built with React.js.

## Features

- ✅ Add rental records (customer, car plate, dates, price per day)
- ✅ Auto-calculates rental days and total cost
- ✅ Filter records by year and month
- ✅ Monthly income bar chart
- ✅ Per-car income pie chart
- ✅ Yearly utilization line chart
- ✅ Export records to CSV
- ✅ Data saved locally in browser (localStorage)

## Tech Stack

- React.js
- Recharts (charts library)
- CSS3

## Getting Started

```bash
npm install
npm start
```
